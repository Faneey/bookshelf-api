# Bookshelf REST API

## Overview

A RESTful API developed using Node.js to manage book collections. The API supports creating, reading, updating, and deleting book data through HTTP requests.

## Features

- Add new books
- Retrieve all books
- Retrieve book details by ID
- Update book information
- Delete books

## Technologies

- Node.js
- Hapi Framework
- JavaScript
- Postman

## API Endpoints

| Method | Endpoint |
|----------|----------|
| POST | /books |
| GET | /books |
| GET | /books/{id} |
| PUT | /books/{id} |
| DELETE | /books/{id} |
